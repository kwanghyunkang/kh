landing 효과 구현

landing : 스크롤이 일정위치가 오면, 동작을 처리해서 우리가 보고자하는 형태로 안착하는 것

  • scroll 이벤트
  • 이미지가 나타나는 방법 : opacity 0~1
  • 나타나는 것을 효과를 좀 더 풍부하기위해서는 transform:transition() || margin-top || top
  • 나타나는 시점은 화면상의 하단에서부터 약 1/3 ~ 1/4에서 나타나야 자연스럽다.
  • mousewheel, DOMMouseScroll 마우스휠에 의해 반응하는 기능 -> 휠값이 아래/위
  • mousewheel -> 120/-120 | DOMMouseScroll -> -3/3
  • 수치를 통일 -> DOMMouseScroll수치 * -40

mousewheel 처리

firefox에서는 DOMMouseScroll 이벤트를 가지고, 이외는 mousewheel 이벤트를 가진다.

이때, 수치는 DOMMouseScroll에서는 -3/3의 수치를 가지고, mousewheel에서는 120/-120의 값을 가지고 있다.

그렇기에 둘의 동일한 수치를 만들고 이를 수행할 수 있도록 처리
